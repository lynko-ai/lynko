# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Lynko, please report it responsibly.

**Do NOT file a public GitHub issue for security vulnerabilities.**

Instead, email **security@lynko.ai** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge receipt within 48 hours and provide an initial assessment within 5 business days.

## Scope

This policy covers the Lynko platform, including:

- MCP endpoints (`mcp.lynko.ai`)
- API endpoints (`api.lynko.ai`)
- Authentication service (`auth.lynko.ai`)
- The Lynko DSL and SDK

## Supported Versions

During early access, we support only the latest deployed version. Security fixes are applied to the current release.
